
# Стек проекта
Backend: Golang/gin/jwt
Frontend: html/css
Database: PostgreSQL
Контейнеризация: Docker,Docker-compose
# Описание проекта
Разрабатывается маркетплейс с поддержкой ролей пользователя и администратора, где реализован полный цикл взаимодействия с товарами и заказами. Пользователь может просматривать товары по категориям, добавлять их в список желаемого или в корзину, оформлять заказ (с получением уникального ID и списка позиций) и пополнять баланс — все операции оформляются как симуляции без реальных транзакций. Также пользователь может отменить заказ, если он ещё не принят администратором, и отслеживать статус заказа (оформлен / принят). Администратор управляет ассортиментом — добавляет и редактирует товары, категории, описания и изображения, а также может просматривать все заказы, видеть их содержимое и менять статус заказа. Главный администратор, помимо этого, имеет возможность добавлять или удалять администраторов, контролируя систему управления.

# API Эндпоинты
**1. POST /user/signup - регистрация нового пользователя**

**Запрос:**
```json
{
    "user_login":"login12",
    "user_password":"lsdDS3221ds"
}
```

**Ответ httpStatus 201 Created:**
```json
{
    "success": {
        "user_id": "f8caf404-aaf7-458e-8f77-3a7d55af2f36",
        "user_login": "login12"
    }
}
```

Ошибки:

| Код | Сообщение | Описание |
|-----|-----------|----------|
| 400 |`incorrect json body`| Некорректный формат JSON|
| 500 |`internal server error`| Ошибка на стороне сервера|
| 409 |`user already exist`|Пользователь уже существует|
